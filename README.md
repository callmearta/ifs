# IFS Guide Voice AI

This is a voice AI for the IFS Guide. It is a simple voice AI that uses the OpenAI API to chat with.

## Development
Make sure to create .env file and fill in the following variables:
```
VITE_OPENAI_API_KEY=
VITE_OPENAI_PROJECT_KEY=
VITE_OPENAI_ORG_KEY=
```

```bash
bun install
bun run dev
```
